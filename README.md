# 🌍 Climate Quiz - Educational Quiz App

## 📝 Descrizione del Progetto
Climate Quiz è un'applicazione web interattiva sviluppata con React e TypeScript, progettata per sensibilizzare le persone sul tema dell'ambiente e del cambiamento climatico. L'app offre un'esperienza di quiz coinvolgente e competitiva, permettendo agli utenti di testare le proprie conoscenze mentre imparano fatti importanti sul nostro pianeta.

## 🌟 Caratteristiche Principali

### Sistema Quiz
- Quiz interattivo con domande a scelta multipla
- Timer per ogni domanda (30 secondi)
- Sistema di punteggio dinamico
- Feedback immediato per le risposte
- Diverse modalità di gioco (Facile, Normale, Difficile)

### Sistema Achievement
- Achievement sbloccabili basati su performance
- Popup di notifica per gli achievement ottenuti
- Salvataggio dei progressi in locale

### Interfaccia Utente
- Design responsive per desktop e mobile
- Animazioni fluide tra le domande
- Feedback visivo per le risposte
- Barra del tempo dinamica
- Sistema di classificazione finale

## 🛠 Tecnologie Utilizzate
- React 18
- TypeScript
- Framer Motion (per le animazioni)
- CSS Modules
- Local Storage (per il salvataggio dei progressi)
- Jest (per i test)

## 📋 Prerequisiti
- Node.js (versione 14.0.0 o superiore)
- npm o yarn

## 🚀 Installazione e Avvio

```bash
# Clona il repository
git clone [https://github.com/arbone/climatequizappTS]

# Naviga nella directory del progetto
cd climate-quiz

# Installa le dipendenze
npm install

# Avvia l'applicazione in modalità sviluppo
npm start
```

L'applicazione sarà disponibile all'indirizzo `http://localhost:3000`
Oppure deployato: climate-quiz.netlify.com


## 🎮 Modalità di Gioco

### Facile
- Tempo limite: 45 secondi per domanda
- Moltiplicatore punti: x1
- Ideale per principianti

### Normale
- Tempo limite: 30 secondi per domanda
- Moltiplicatore punti: x2
- Bilanciato per la maggior parte degli utenti

### Difficile
- Tempo limite: 15 secondi per domanda
- Moltiplicatore punti: x3
- Sfida impegnativa per esperti
